This site is for demonstration only. Please run Gallica Autobib locally for full functionality. Or roll your own server: this code for this demonstration is on Github. It is written in FastAPI and uses a centerally managed pool of concurrent.futures.ThreadPoolexeuctor processes to handle requests.
